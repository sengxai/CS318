Display Block

Block elements display one after the other, starting at the top and moving down the page. Block elements appear on the screen as if they have a line break before and after them.

The header, main, and navigation tags are all also display block by default. As well as the section, article, and footer tags that we will be using in lab 5.

Div is a block level non-semantic element, which you should only use if you can't think of a better semantic block element to use, or don't want to add any specific meaning. Since it is a block element it will create a element underneath the previous element.
