Classes and IDs

Class and id attributes are used to label elements on a finer scale than the element name.

The difference between classes and IDs is that mutliple tags can have the same class but never the same id.

HTML tags can have mutliple classes. To do this you use the same class attribute but have multiple values separated by spaces.
